API Endpoints :

GET /fetch/
DELETE /<id>/delete/
PUT /<id>/update/
POST /create/

Method	 Endpoint	     Description
GET	    /fetch/	        Retrieve all tasks
POST	  /create/	     Create a new task
PUT	   /{id}/update/	 Update an existing task
DELETE  /{id}/delete/	  Delete a task
